⚙️ Procedimiento: Valorización de Inventario

Este procedimiento calcula el valor total del inventario sumando el precio de cada componente multiplicado por su stock disponible:


Valor Total = Σ (Precio × Stock) para todos los componentes

⚠️ NOTA: Este endpoint NO es REST porque representa una acción/procedimiento, no un recurso (sustantivo plural).
